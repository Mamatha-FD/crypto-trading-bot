Binance Futures Testnet – Simplified Trading Bot

Overview
This project is a Python-based trading bot built for the Binance Futures Testnet.
It supports Market, Limit, and Stop-Limit orders via a command-line interface using the official Binance API.

Features
- Market Order
- Limit Order
- Stop-Limit Order (Bonus)
- Buy and Sell support
- CLI-based interaction
- API logging and error handling

Tech Stack
- Python 3
- python-binance library
- Binance Futures Testnet

Project Structure
screenshots/ – Execution screenshots
first.py – Main bot script
README.docx – Documentation
trading_bot.log – Logs


Installation
1. Install Python 3
2. Install dependencies: pip install python-binance
3. Create Binance Futures Testnet account
4. Generate API keys

How to Run
python first.py

Logging
All API activities are logged in trading_bot.log.
